# H E L L O  W O R L D

This is a simple Task taking App. Feel free to 
use and modify as your need.

### Technology i used
    - HTML5
    - CSS3
    - Vanilla JavaScript
    - Local Storage/Browser Storage

### Feature of this App
    1. Change Theme (Dark, Light)
    2. Add your task/todo
    3. Delete your task/todo
    4. Filter your todo
        - All
        - Completed
        - Active
    5. You can reorder task list
